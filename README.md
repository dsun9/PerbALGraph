# PerbALGraph

The code for paper Perturbation-based Graph Active Learning for Semi-Supervised Belief Representation Learning. The dataset is not yet released due to privacy issue. We will consider publishing datasets when policy permits.

The format of the dataset is shown in a sample CSV file. the model can be run by the following command:
```
python perb_al_graph.py
```

Configurations can be modified in `conf/config.yaml`.
